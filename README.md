# Border Radius Previewer

A project idea from [florinpop17](https://github.com/florinpop17/app-ideas/blob/master/Projects/1-Beginner/Border-Radius-Previewer.md)

Border Radius Previewer is an interactive tool for experimenting with the CSS border-radius property. Adjust the radius of each corner using sliders and instantly see the resulting shape and generated CSS.

## User Stories

-   [x] User can see a box which has a `border-radius` property applied to it
-   [x] User can change the 4 `border-radius` values that are applied to the box (top-left, top-right, bottom-left, bottom-right)
-   [x] User can copy the resulting CSS to the clipboard
